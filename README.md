A Forum System
======

It developed with node.js and it uses mongodb.


add a file that named 'config.js'
```  
  exports.config=
	{
		kk_url		 	:"http://localhost:8080",
		port 		 	:"8080",
		SiteName	 	:"njsforum",
		DBConnection 	:"localhost:27017/davet",
		consumer_key 	:"secret",
	    consumer_secret :'secret',
	    user 			: user_email,
	    passwd          : passwd  
	}
```
